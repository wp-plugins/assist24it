== Installation ==

**Installation**

*Install and activate plugin 'Assist24'
*Click on the new A24 button in your left sidebar.
*Click the button to take you to the widget selector at 'A24 Plugin Options'.
*Enter your 'A24 Room ID' or create a new account (app.assist24.it).
*Click the link 'Save Setting' and get ready to chat!

**Manual Installation**
1. Download the plugin 'el-assist24' from www.assist24.it,  Upload and Activate
2. Visit the "Assist24.it" options page, enter your Assist24.it ID, and done.

For more information, visit the [A24 Plugin Homepage](http://www.Assist24.it/wordpress-plugin).
